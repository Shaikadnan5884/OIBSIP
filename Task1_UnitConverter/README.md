# 📏 Professional Unit Converter App

**OIBSIP Internship - Task 1**

A high-performance, visually stunning Android application built with Java and XML. This app isn't just a simple converter; it's a specialized tool featuring a custom calculator-style interface designed for speed, precision, and a premium user experience.

---

## ✨ Key Features

- **9 Conversion Categories**: Comprehensive support for:
  - 📐 **Length**: Centimeters, Meters, Kilometers, Inches, Feet, Miles
  - 🗺️ **Area**: m², km², cm², ft², Acres, Hectares
  - 🧪 **Volume**: Liters, Milliliters, Cubic Meters, Gallons, Cubic Feet
  - ⚖️ **Weight**: Grams, Kilograms, Milligrams, Pounds, Ounces
  - 🌡️ **Temperature**: Celsius, Fahrenheit, Kelvin
  - 🚀 **Speed**: m/s, km/h, mph, Knots
  - 💨 **Pressure**: Pascal, Bar, Atmosphere, PSI
  - ⚡ **Power**: Watts, Kilowatts, Horsepower
  - 🔢 **Number Systems**: Decimal, Binary, Hexadecimal, Octal
- **Real-Time Calculation**: Results update instantly as you type on the custom keypad.
- **Unit Swap**: Quickly switch "From" and "To" units with a single tap.
- **Error Resilience**: Robust handling of invalid numeric inputs and edge cases.

---

## 🎨 UI/UX Excellence

- **Modern Dark Aesthetic**: A deep `#121212` background paired with neon green accents for a high-tech feel.
- **Triple-Tone Gradients**: Beautiful green gradients (**Neon Green** → **Mint** → **Teal**) on the Toolbar and primary Action Buttons (AC/DEL).
- **Custom Keypad**: A built-in calculator-style number pad for a specialized, faster input experience without the system keyboard.
- **Smooth Animations**: Interactive `StateListAnimators` provide satisfying scale-down and elevation feedback on every button press.
- **Custom Vector Icons**: Unique, category-specific icons (Rulers, Beakers, Gauges) for an intuitive navigation grid.
- **Polished Card Design**: Angled dark-grey gradients on the category selection screen for a premium "brushed metal" depth.

---

## 🛠️ Tech Stack

- **Language**: Java
- **Layouts**: XML (ConstraintLayout, GridLayout)
- **UI Components**: CardView, Material Buttons, Custom Vectors, Ripples, and Gradients.
- **Logic**: Centralized base-unit conversion engine for maximum mathematical accuracy.

---

## 🚀 Installation & Usage

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/Shaikadnan5884/OIBSIP/tree/main/Task1_UnitConverter
   ```
2. **Open in Android Studio**: Import the project from the `Task1_UnitConverter` folder.
3. **Run**: Deploy to an emulator or physical device.
4. **Convert**: Choose a category from the home grid, type your value on the keypad, and see the result instantly!
